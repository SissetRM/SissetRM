### 3D Diffusion Limited Accretion Simulation (Python, NumPy, Matplotlib)

- Developed a 3D stochastic particle aggregation simulation modelling diffusion-limited fractal growth.
- Implemented a Monte Carlo random walk algorithm in three dimensions, where particles are initialised on a spherical boundary and iteratively move until attaching to an existing cluster.
- Dynamically resized the growth boundary as the lattice expanded, enabling simulation of 10,000+ particle structures.
- Computed aggregation using minimum squared Euclidean distance comparisons, resulting in approximate O(N²) time complexity.
- Generated 3D visualisations with colour-mapped radial gradients and exported coordinate datasets for structural analysis.
- Demonstrates computational modelling, algorithmic design, numerical simulation, and performance analysis in scientific programming.
